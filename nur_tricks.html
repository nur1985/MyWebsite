c++ Tricks

  • Change text color of a terminal
Define the color first and use it in a C file. Here is an example file named terminal_color.C. Run it as root -l terminal_color.C. char red[] = { 0x1b, '[', '1', ';', '3', '1', 'm', 0 }; char green[] = { 0x1b, '[', '1', ';', '3', '2', 'm', 0 }; char blue[] = { 0x1b, '[', '1', ';', '3', '4', 'm', 0 }; char magenta[] = { 0x1b, '[', '1', ';', '3', '5', 'm', 0 }; char normal[] = { 0x1b, '[', '0', ';', '3', '9', 'm', 0 }; void terminal_color(){ std::cout<<Form("What color is your terminal")<<std::endl; std::cout<<Form("%s Blue %s",blue,normal)<<std::endl; std::cout<<Form("%s Red %s",red,normal)<<std::endl; std::cout<<Form("%s Green %s",green,normal)<<std::endl; }

    • Ans: Text1<span style="padding-left:100px">Text2</span> Result: Text1Text2

    • Ans: <marquee direction="left" behavior="alternate" bgcolor=transparent width=274 height=20 scrollamount=4 style="border: dotted 1px #888888;"onmouseover="this.stop();" onmouseout="this.start();">Text to scroll.</marquee> Result: Text to scroll.

Unix Commands

  • Change file permision in Unix using chmod.

In a terminal do

chmod 755 filename to change the permission to read-write-execute for user, read-write for group and read-write for world. See other examples in the table.

User Group World
r+w+x r+w+x r+w+x
4+2+1 4+2+1 4+2+1 777
r+w+x r+x r+x
4+2+1 4+0+1 4+0+1 755
r+w+x r r
4+2+1 4+0+0 4+0+0 744
r r+w r+w+x
4+0+0 4+2+0 4+2+1 467

  • Change the user and/or group ownership of a given file or directory in Unix.
chown options owner-user:owner-group file Example: chown -R nur:minerva work Changed the owner of the directory "work" recurcievely. The user is changed to "nur" with group "minerva". You have to login as root to do these changes.
